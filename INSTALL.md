# 安装指南

## 快速开始

### 1. 准备图标（可选）

如果您有图标文件，请将它们放在 `icons/` 目录下：
- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)
- `icon128.png` (128x128 像素)

如果没有图标，扩展仍然可以正常工作，Chrome 会使用默认图标。

### 2. 在 Chrome 中加载扩展

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在右上角开启"开发者模式"开关
4. 点击左上角的"加载已解压的扩展程序"按钮
5. 选择本项目文件夹（`bookmarks_deepseek_extension`）
6. 扩展加载成功后，您会在扩展栏看到扩展图标

### 3. 配置 API 密钥

1. 点击扩展图标，然后点击右上角的设置按钮（⚙️）
2. 或者右键点击扩展图标，选择"选项"
3. 在设置页面输入您的 DeepSeek API 密钥
4. 点击"保存设置"按钮

**获取 API 密钥：**
- 访问 [DeepSeek 平台](https://platform.deepseek.com)
- 注册/登录账号
- 在 API 管理页面创建新的 API 密钥
- 复制密钥并粘贴到扩展设置中

### 4. 开始使用

1. **分类整理书签：**
   - 点击扩展图标
   - 点击"📖 加载书签"按钮
   - 等待书签加载完成
   - 点击"🤖 AI分类整理"按钮
   - 等待 AI 分类完成，查看分类结果

2. **检测无效书签：**
   - 切换到"检测无效"标签页
   - 点击"🔍 检测书签可访问性"按钮
   - 等待检测完成
   - 查看不可访问的书签列表

## 常见问题

### Q: 扩展无法加载？
A: 确保您已开启"开发者模式"，并且选择了正确的项目文件夹。

### Q: API 密钥无效？
A: 请检查：
- API 密钥是否正确复制（没有多余空格）
- API 密钥是否有效（未过期）
- 网络连接是否正常

### Q: 分类结果不准确？
A: AI 分类基于书签的标题和 URL，如果信息不完整，可能影响分类准确性。您可以尝试：
- 确保书签有清晰的标题
- 重新运行分类功能

### Q: 检测速度很慢？
A: 检测书签可访问性需要逐个访问每个链接，所以速度取决于：
- 书签数量
- 网络速度
- 目标网站的响应速度

建议在书签较少时使用，或分批检测。

## 注意事项

- 检测功能会打开临时标签页，这是正常的
- 请确保网络连接稳定
- API 调用可能会产生费用（取决于您的 DeepSeek 套餐）
- 建议定期备份书签
